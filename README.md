## Local Preview

This is a static website. You can open `index.html` directly in a browser or use a simple local server:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deployment

The site is configured for GitHub Pages with a custom domain defined in `CNAME`.
